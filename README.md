# 🦉 Estratégia PDF Downloader

Script em Python para automatizar o download de materiais (PDFs - Versão Original) do Estratégia Concursos.

## 🚀 Funcionalidades
- Faz login automático via sessão salva (evita captchas repetitivos).
- Identifica os títulos das aulas automaticamente.
- Salva os arquivos organizados por pastas na sua pasta de Downloads.
- Compatível com GitHub Secrets para automação futura.

## 🛠️ Como Instalar
1. Clone o repositório: `git clone https://github.com/seu-usuario/seu-repositorio.git`
2. Crie um ambiente virtual: `python -m venv venv`
